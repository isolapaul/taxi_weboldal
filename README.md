# 🚕 Patkó Taxi Balatonboglár

A clean, fast, and fully responsive website built for a local taxi service operating in Balatonboglár, Hungary. 

🌐 **Live Website:** [taxipatkobalatonboglar.hu](https://taxipatkobalatonboglar.hu)

## ✨ Features

- **Multilingual Support:** Instant language switching between Hungarian (HU) and German (DE) without page reloads.
- **Mobile-First Design:** Looks great on all devices. Features a smart, auto-hiding navigation bar that disappears when scrolling down to maximize screen space and reappears when scrolling up.
- **Quick Contact Options:** Floating action button for instant phone calls and a dedicated WhatsApp integration.
- **Smooth Navigation:** Hamburger menu on mobile with smooth scrolling to different sections (Services, Reviews).
- **Google Maps Integration:** Direct links to the business's Google Maps location for easy navigation and a dedicated button to leave a review.
- **SEO & Social Sharing Ready:** Includes Open Graph (OG) meta tags so the website looks great with a proper preview image when shared on social media and messaging apps.

## 🛠️ Tech Stack

This project is intentionally kept lightweight and fast. No heavy frameworks were used.
- **HTML5** (Semantic structure)
- **CSS3** (Custom variables, Flexbox/Grid, Animations, Responsive media queries)
- **Vanilla JavaScript** (Language switching, smooth scroll handling, mobile menu toggle)
- **FontAwesome** (Icons)
